# ☕ Java Programs

A collection of Java programs covering basic to intermediate concepts, designed to strengthen core programming skills and prepare for technical interviews. Useful for beginners, students, and anyone revisiting Java fundamentals.

---

## 📚 Topics Covered

| Category | Programs Include |
|----------|-----------------|
| **Basics** | Hello World, data types, operators, type casting |
| **Control Flow** | if-else, switch, loops (for, while, do-while) |
| **Arrays** | Array traversal, sorting, searching, 2D arrays |
| **Strings** | Palindrome, anagram, reverse, string methods |
| **Methods & Recursion** | Factorial, Fibonacci, power, GCD |
| **OOP** | Classes, objects, inheritance, polymorphism, interfaces |
| **Pattern Programs** | Star patterns, number patterns, pyramid |
| **Problem Solving** | Number series, prime numbers, Armstrong, perfect numbers |

## 🚀 Getting Started

### Prerequisites
- Java JDK 8 or higher
- Any IDE (IntelliJ IDEA, Eclipse, VS Code) or terminal

### Run a Program

1. **Clone the repository**
   ```bash
   git clone https://github.com/alekhyaGontla/Java_Programs.git
   cd Java_Programs
   ```

2. **Compile and run**
   ```bash
   javac FileName.java
   java FileName
   ```
